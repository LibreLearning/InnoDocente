\begin{frame}
\frametitle{Y al principio fue el código...}

Prácticas de la comunidad del software libre:

{\Large
\begin{itemize}
\item ``Muéstrame el código fuente''
\item Aprendizaje leyendo el código de otros
\item Aprendizaje modificando, incluso estropeando el código de otros
\item Co-aprendizaje trabajando en el mismo código
\item Aprendizaje escribiendo código, y recibiendo críticas
\item Mejora incremental por revisión de pares
\end{itemize}
}
\end{frame}

%%---------------------------------------------------------------

\begin{frame}
\frametitle{El patrón}

{\Large
Primero leer, luego modificar, luego escribir
\begin{flushright}
...y aprender mientras tanto.
\end{flushright}

\vspace{.5cm}

Escuchado a muchos educadores:

\begin{itemize}
\item cuando lees, aprendes algo
\item cuando escribes, aprendes más
\item cuando explicas, es cuando aprendes de verdad
\end{itemize}
}
\end{frame}

%%---------------------------------------------------------------

\begin{frame}
\frametitle{La idea}

{\Large
Que los alumnos mejoren su aprendizaje escribiendo apuntes en colaboración

\begin{itemize}
\item Los apuntes de los alumnos pueden ser los mejores apuntes
\item Alta motivación para escribir buenos apuntes para ellos mismos
\item Los alumnos de la misma asignatura escriben sobre los mismos temas
\item Los portátiles, las grabadoras facilitan un proceso completamente digital 
\end{itemize}

}

\end{frame}
